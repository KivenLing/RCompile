输入：String[]（String数组），例如：a+b -> c 变成 a  +  b  -  >  c
输出：WordMean类
WordMean：一个List<Token>变量，
Token类：word和wordmean（都是String）
					其中word是单词拆开，wordMean是单词类型
					wordMean有几类，符号类symbol（直接字符串），
					虚数:CommonVar.COMPLEX = "complex";
					数字：CommonVar.NUMERIC = "numeric";
					TRUE FALSE：CommonVar.LOGICAL = "logical";
					整形：CommonVar.INTEGER = "integer";
					字符串：CommonVar.CHARACTER = "character"; 					例如本来是“asd”，变成了asd 和 character
					通用字符：CommonVar.COMMONVAR = "commonvar"; 					变量或者关键字
					隐藏变量：CommonVar.HIDEVAR
